Procesador 8088/86

El procesador 8088/86 utiliza registros de 16 bits. Hay 14 registros en total, cada uno de ellos con un objetivo especial y accesible a nivel de programación.

  • 4 registros de datos.

  • 4 registros de segmentos.

  • 2 registros puntero de pila.

  • 2 registros de indice.

  • 1 registro puntero de instrucciones.

  • 1 registro de estado.

Registro de Datos y Segmentos

Registro de Datos

Hay 4 registros de 16 bits llamados REGISTROS DE DATOS de almacenamiento temporal o de trabajo. Estos registros se conocen como AX, BX, CX y DX. Cada uno de ellos, a su vez, pueden subdividirse y direccionarse de forma separada como si fueran dos medios registros de 8 bits. Los registros de mayor orden se denominan AH, BH, CH y DH, y los de orden bajo AL, BL, CL y DL.

Cada uno con un uso especial asignado:

  • AX: utilizado en las instrucciones aritmeticas.
  • BX: se usa para indicar un desplazamiento.
  • CX: contador para control de bucles y operaciones de tipo repetitivo.
  • DX: almacenador de datos de 16 bits con propositos generales.
Registro de Segmentos

Estos 4 registros, llamados CS, DS, SS y ES, se utilizan para identificar 4 segmentos de 64k especificos de memoria. Es decir, contienen la direccion de comienzo de ciertos segmentos de memoria.

  • CS: contiene la dirección a partir de la cual comienza el programa que se esta ejecutando.
  • DS: contiene la dirección donde empieza el area de memoria en la que se encuentran almacenados los datos que utiliza el programa.
  • SS: contiene la dirección donde empieza un lugar de trabajo temporal que almacena parametros y direcciones que utiliza el programa.
  • ES: segmento extra de datos adicional que se utiliza para superar la limitacion de 64k del segmento de datos y para hacer transferencia de datos entre segmentos.
